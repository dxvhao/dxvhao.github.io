# 集群登陆

提示

集群管理软件默认的帐号与密码均为 admin

# 本机登陆

  • 在桌面找到 集管快捷方式 双击运行，若已经运行在任务栏中找到 集管图标 双击可打开集管到登陆页面
  • 打开浏览器，在地址栏输入本机IP地址 + 端口 8000 例：127.0.0.1:8000

# 内网远程登陆

使用浏览器输入 xxx.xxx.xxx.xxx:8000 远程访问集群管理, xxx 请替换为你安装集群软件的计算机的IP地址。

提示

为了可以正常远程/本地访问管理页面，建议使用 Chrome浏览器 。并且注意端口之间的冒号使用英文输入法输入。

# 如何将主机加入集群

# 进入主机后台管理系统

  • 重启主机，在显示多路云版本号信息时，按 Ctrl 键进入主机后台管理系统
  • 使用浏览器在地址栏输入后台ip地址 xxx.xxx.xxx.xxx:3000 远程访问主机后台管理系统，请将 xxx 替换为主机的IP地址

# 加入集群

  • 在主机后台管理系统中，进入 主机管理 选项卡，单击 搜索可用 按钮，在搜索到的集群信息中，选择要加入的房间，单击 请求加入 按钮，最后确认编号无误后，单击 加入 按钮，完成集群加入。

提示

如果"搜索可用"按钮无法搜索到集群服务，可单击 "手动加入" 按钮，手动输入集群服务IP地址与端口来加入，格式： xxx.xxx.xxx.xxx:8000

警告！

不要将重复的终端编号加入到集群，否侧在 windows 配置时会出现错误。推荐按照顺序一台一台主机加入集群。

# 配置源主机

打开集群软件，在左侧栏中展开 主机管理 单击 常用功能 在主机列表中的标题对应的后台图标来进入主机的后台管理系统。

# 导入镜像

在后台管理页的左侧栏中选择进入 外部存储 选项卡，单击 USB存储 按钮，在 U盘文件 列表中单击 加载 按钮。在U盘文件列表中选择镜像导入，等待完成！

提示

如无法加载出U盘时，请检查主机是否进入维护状态，如果未进入维护状态，请在主机管理 - 常用功能中单击进入维护状态！

# 扩容镜像

警告！

镜像扩容操作是不可逆的，一旦扩容成功就不可撤回！如果你有撤回的需求，建议可在第二层子镜像扩容。

镜像导入成功后，单击侧边栏中的 镜像数据 找到刚刚导入的镜像，单击 扩容 图标按钮进行扩容。

提示

请根据您的物理硬盘的实际容量来扩容增加父镜像的容量。如想扩容50GB 当前镜像为30GB 就添加20GB 即可。

# 使用集群 新建系统盘模板 同传向导

进入集群，在左侧栏中展开镜像管理，单击新建和更新模板选项，选中 新建系统盘模板 再次选择房间，最后单击 下一步 按钮。

# 将所有机器进入维护状态

单击 开始 按钮，等待集群中所有机器进入 维护状态 ，成功进入维护状态后，下一步 按钮会激活。

# 生成模板

这里的主机要选择我们之前导入镜像的机器 勾选1号机 ,为 模板镜像名 起个名字，磁盘大小 建议 基础镜像 + 所有子镜像 不要超出物理硬盘的的总和。 选择 分配到已有菜单 选择默认菜单 win7 单击 下一步

# 编辑镜像

集管会为您打开一个终端，请留意1号机的1号终端会被打开，此时操作终端进行安装软件、或扩盘等操作。

# 扩展windows磁盘

只要在集群或者主机后台管理中操作过扩容镜像盘的操作，就要在终端用户中进行扩展windows磁盘。

提示

在 新建系统盘模板 向导中的 生成模板 中配置 磁盘大小 并在编辑镜像模式中 扩展windows磁盘 会将容量扩容继承给所有终端哦。

# 打开计算机管理

在 计算机 单击 右键 选择 管理 打开计算机管理。

提示

若非 Win7 操作系统，或桌面并没有显示 计算机、此电脑 可在键盘中按 Win 键 + E 键 在弹出的窗口的左侧栏中 右键 -> 管理 来打开计算机管理

# 打开磁盘管理

在计算机管理中展开 存储 选择 磁盘管理 在 磁盘 0 （C:） 右键选择 扩展卷 接下来点击 下一步 直到完成为止。

# 编辑完成

这里不在演示安装软件的过程，回到集群单击 编辑完成 向导会自动关闭终端并且恢复到 维护状态

提示

一个简单的概念，向导中的 编辑镜像 是指修改镜像，像安装软件、修改Windows配置等都称做编辑镜像，向导接下来会指引您将这个编辑好的镜像 同传 给所有 目标主机

# 选择目标主机

源主机 即是发送方，而 目标主机 即接收方，单击如下图中的选择框为 全选 随后单击 下一步

# 选择要克隆的内容

向导默认选择 启动菜单 首次部署默认配置就可以，单击 下一步

# 主机检查

检查主机是否正常，如果存在异常请检查，正常的数量是否一致。

# 同传配置

同传源主机的配置

# 同传镜像

下图中标记位置会有 开始同传 的按钮，单击后会进行镜像同传，同传完成后，按钮会消失。

# 一致性检查

会显示同传的结果，与同传镜像的数量。

# 终端检查

终端检查 会将所有主机退出 维护状态 并重启主机并且启动所有 终端 。 请检查所有终端是否启动成功，并且无任何问题，最后单击 检查完成 按钮。

# Windows配置

  • 如果你导入的镜像是由我们提供的，可以跳过这里。 如果镜像为你自己安装的包，请先确定镜像是否安装了 btvmtool 客户端。

  • 在使用Windows配置前，请确认终端已全部关机了。 可进入 主机管理 -> 常用功能 来查看 终端 的开关状态。

# 批量配置静态IP地址

在集群左侧栏中单击 Windows配置 进入Windows配置面页，单击 批量设为静态IP 来批量配置房间内所有终端的静态IP地址。 先单击 预览 按钮，再单击 提交 按钮

# 批量设置计算机名

单击 批量设置计算机名 按钮，在 前缀 中输入要设置的计算机名，集管会为前缀的后缀追加3位从1开始的数字。如下图

# 批量设置Win用户名

Win用户名 就是Windows操作系统的帐户名，批量设置的方法与 批量设置计算机名 类拟，但是值得注意的是，将 前缀 设置为 Administrator 后两项表单用 退格键 删掉，集群会将所有终端设置为 Administrator

# 还原

  • 开启 自动还原 前要确定 终端 是否已关闭。

提示

自动还原开启后，只有终端开关机才会自动还原，重启终端不会进行自动还原！

# 批量开启自动还原

通过集群侧边栏展开 镜像管理 -> 常用操作 选中 自动还原 操作方式选择 批量 然后单击 启用自动还原 完成开启自动还原
